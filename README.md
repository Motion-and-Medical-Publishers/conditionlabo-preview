## デザイン仕様

- デザインパターン: Split Cinematic（Pattern M）
- カラー: ダークグリーン（#0D1F15）+ アクセントグリーン（#4FA97A）
- フォント: Cormorant Garamond / Noto Serif JP / Noto Sans JP
- レスポンシブ対応: Desktop (1025px+) / Tablet (769-1024px) / Mobile (768px以下)

## 主な機能

- ヒーローセクション（PC/SP 画像切替）
- スタッフプロフィールモーダル × 5名
- 施術メニュー詳細モーダル
- 著書一覧モーダル（30冊+）
- 患者様の声モーダル
- 行き方ガイドモーダル（4ステップ）
- Google Maps 埋め込み（クリック後ロード）
- YouTube 動画 × 4本（クリック後ロード、lite-embed）
- お問い合わせフォーム（mailto 方式）
- 運営会社モーダル
